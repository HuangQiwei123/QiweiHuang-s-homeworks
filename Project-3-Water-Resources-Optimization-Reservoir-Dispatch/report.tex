\section*{Objective}
This project optimizes seven-day reservoir releases under hydropower revenue, ecological release, storage, and mass-balance constraints.
\section*{Optimization}
The implementation uses SciPy SLSQP when available and a linear fallback solver in lightweight environments. The objective is to compare revenue-oriented and ecology-oriented operating policies.

\section*{Validation}
The validation report checks release bounds, storage bounds, and daily mass balance. The generated hard-ecology solution satisfies all physical constraints.
