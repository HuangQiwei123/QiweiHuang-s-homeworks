\section*{Abstract}
This experiment implements the Soil Conservation Service Curve Number (SCS-CN) method for estimating direct runoff from rainfall depth. The work translates hydrological equations into tested Python code, verifies boundary conditions, supports vectorized NumPy inputs, and visualizes runoff sensitivity to curve number and rainfall depth.

\section*{Objectives}
\begin{itemize}
  \item Implement the SCS-CN formula using clear Python functions.
  \item Encode physical boundary conditions, including no-runoff and impervious-surface limits.
  \item Validate that runoff never exceeds rainfall.
  \item Produce sensitivity plots for different curve numbers.
\end{itemize}

\section*{SCS-CN Method}
For rainfall depth $P$ and curve number $CN$, the potential maximum retention is
\[
S = \frac{25400}{CN} - 254.
\]
The initial abstraction is
\[
I_a = 0.2S.
\]
When $P \le I_a$, direct runoff is zero. Otherwise,
\[
Q = \frac{(P-I_a)^2}{P-I_a+S}.
\]

\section*{Boundary Conditions}
\begin{longtable}{ll}
\toprule
Condition & Implemented behavior \\
\midrule
$P < 0$ & Raise a value error \\
$CN < 0$ or $CN > 100$ & Raise a value error \\
$CN = 0$ & Complete infiltration, $Q=0$ \\
$CN = 100$ & Impervious surface, $Q=P$ \\
$P \le I_a$ & No direct runoff, $Q=0$ \\
$Q > P$ & Clip to $P$ for physical consistency \\
\bottomrule
\end{longtable}

\section*{Implementation}
The function \texttt{calculate\_runoff(P, CN)} accepts scalar values or NumPy arrays. Broadcasting allows rainfall and curve number combinations to be evaluated efficiently. The test file \texttt{test\_scs\_cn.py} covers zero precipitation, the initial abstraction threshold, normal cases, the maximum curve number, vectorized inputs, and invalid parameters.

\section*{Sensitivity Analysis}
The script \texttt{sensitivity\_analysis.py} fixes rainfall at 50 mm for one plot and varies rainfall depth from 0 to 100 mm for another plot. The result shows that high curve numbers generate much larger runoff because retention and initial abstraction decrease sharply as imperviousness increases.

\section*{Results}
For the reference case $P=50$ mm and $CN=80$, the computed runoff is approximately 13.8 mm, matching the expected hand calculation. The figure \texttt{scs\_cn\_sensitivity.png} demonstrates the nonlinear increase in runoff with curve number and the limiting behavior $Q=P$ at $CN=100$.

\section*{Validation and Interpretation}
The validation focuses on hydrological plausibility. Low curve numbers delay runoff onset, while high curve numbers increase flood response. The implementation enforces $Q \le P$, which prevents nonphysical results from formula misuse or numerical edge cases.

\section*{Limitations and Future Work}
The SCS-CN model is event-based and does not route runoff through a watershed. Future extensions could include antecedent moisture condition adjustment, time-area routing, or comparison with the Rational Method.
