\section*{Objective}
This project builds a rainfall monitoring dashboard that fetches or simulates city-scale rainfall intensity, classifies warning levels, logs red alerts, and visualizes results through Streamlit and Folium.
\section*{Implementation}
The core script is \texttt{weather\_monitor.py}. It supports OpenWeatherMap data, simulation fallback, alert-level classification, timestamped alert logs, historical data export, and a five-minute refresh cadence.

\section*{Validation}
The system checks threshold behavior, handles API failures gracefully, and keeps rainfall values in physically reasonable units of mm/h.
